Como importar los archivos del repositorio de objetos

A.      Primer Paso: Seleccionar la fuente de los repositorios de objetos.

 

Gitlab: La herramienta leer los repositorios de objetos(OR) desde Gitlab a travs del API.

Manual OR upload: Usted debe importar los OR que tiene en el directorio /objectrepository.

 

B.      Si selecciona la opcin Gitlab, deber seleccionar las siguientes opciones.

1.       Nombre del proyecto

2.       El nombre de la rama(Branch) donde se encuentran los OR que desea utilizar

3.       Sistema operativo

a.       Web: OR en el directorio keyword-test-automation-framework

b.       iOS: OR de iOS en el directorio mobile-keyword-test-automation-framework

c.       Android: OR de Android en el directorio mobile-keyword-test-automation-framework

d.       Other Mobile Pages: OR que no se logr identificar [iOS o Android] en el directorio mobile-keyword-test-automation-framework

4.       Seleccionar el nombre del OR (pgina) que desea utilizar

 

 

 

 

Una vez haya seleccionado todas las opciones, la herramienta le mostrar los objetos que se encuentran en dicho repositorio de objetos.

C.       Si selecciona la opcin Manual OR upload:

 

1.       Oprimir botn Import OR

 

2.       Presione Click to choose a file

3.       Abrir la ventana para seleccionar el directorio y los archivos (OR) que desea utilizar.

a.       Nota: Mientras presiona la tecla SHIFT, podr seleccionar mltiples archivos.

 

4.       Una vez importado los archivos, la herramienta interpretar el sistema operativo al que pertenecen.

a.       De igual manera, deber seleccionar el sistema operativo (OS) y luego la pgina que desea utilizar.

Call To Actions (CallTo)

La herramienta interpretar como un Call To Action Feature File a todos los Feature Files que estn localizados en una de estas dos localidades:

1.       Dentro de carpeta CallToActions

2.       Justo dentro de la carpeta de features, pero no dentro de sub-carpetas

 

Nota: Ver imgenes y recuadros en verde

 

Cmo agregar el nombre del CallToAction?

1.       Agregar un paso vaco(Add empty step)

2.       Seleccionar keyword CallTo

3.       Oprimir sobre la celda de Input

 

 

4.       Para ver los pasos del CallToAction, oprimir sobre el botn Show CallTo Steps

Se mostrar una ventana con los pasos del CallToAction

 

 

 

 

 

 

Visualizacin de Locator ( xpath, id etc)

La herramienta tiene la opcin de visualizar tanto el nombre del objeto como el valor del Localizador(xpath, id etc). Esto ayuda para la mejor interpretacin del objeto en casos donde el nombre del objeto no sea suficiente.

 

 

 

 

 

 

Importar Feature File

Es posible importar un Feature File ya existente y aadir nuevas instrucciones al igual que eliminar instrucciones existentes. Nota: Esto no altera el feature File original ya que los cambios se realizan en la interface web y no en el .xls original.
